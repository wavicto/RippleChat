Appendix A.3. MIDDLE DOT
Between ‘l’ U+006C characters only, used to permit the Catalan character ela geminada to be expressed.

§Arguments

  • s: String value to check
  • offset: The position of the character in the label

§Returns

Return true if context permits a MIDDLE DOT U+00B7.
