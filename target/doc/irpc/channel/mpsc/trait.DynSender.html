fn try_send( &self, value: T, ) -> Pin<Box<dyn Future<Output = Result<bool, SendError>> + Send + '_>>

Try to send a message, returning as fast as possible if sending is not currently possible.

For the remote case, it must be guaranteed that the message is either completely sent or not at all.
